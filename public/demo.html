✨ Cómo usamos la IA en este proyecto

💻
Desarrollo del hubTodo el código HTML, CSS y JavaScript fue generado y depurado con asistencia de Claude AI — sin conocimientos previos de programación web.
🤖
Respuestas automáticasLa IA genera las respuestas a tickets de escuelas según el módulo afectado, reduciendo el tiempo de respuesta de minutos a segundos.
📋
Documentación funcionalEl plan de testing, los criterios de aceptación, las historias de usuario y los diagramas de flujo fueron generados por IA desde la documentación existente.
🔗
Integración de sistemasLa IA diseñó la arquitectura para conectar Mantis, BA Colaborativa, Google Apps Script y GitHub sin presupuesto de infraestructura.
📊
Dashboard de métricasEl dashboard de carga de escuelas fue construido completamente con IA a partir de los datos exportados del sistema, reemplazando Power BI.
🎯
Aprendizaje continuoLa IA nos acompañó como asistente técnico en cada decisión, explicando conceptos y proponiendo soluciones adaptadas a nuestro contexto.
